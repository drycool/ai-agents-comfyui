# Инструкция по сохранению сессий

## Быстрые команды

### В конце каждой сессии:
```bash
# 1. Обновить progress.md (добавить что сделано)
# 2. Обновить decisions.md (если есть решения)
# 3. Коммит с контекстом
git add memory/ && git commit -m "Обновление прогресса"
git push
```

### Команда автосохранения:
```bash
# Сохранить все изменения + прогресс
git add . && git commit -m "Сессия X: что сделано"
```

## Структура progress.md

```markdown
## Активная сессия
**Дата:** YYYY-MM-DD
**Задача:** Название задачи

### Что сделано
- [x] Пункт 1
- [x] Пункт 2

### Следующие шаги
- [ ] Пункт 1
```

## Структура decisions.md

```markdown
## YYYY-MM-DD

### Название решения
- **Решение:** Кратко
- **Детали:** Подробности
```
